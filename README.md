# RS_Code_MATLAB

本项目基于 MATLAB 实现 RS 码编码与译码仿真。

## 项目简介

本项目主要完成 RS 码的编码、译码以及仿真分析，通过 MATLAB 编程实现 RS 码编译码过程，并利用 Git 和 GitHub 进行团队协作开发与版本管理。
